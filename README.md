🛠️ Key Modules & Features
🔐 1. User Authentication & Authorization

Multi-role authentication: User, Recruiter, Admin

BCrypt password encryption

Role-based access control (Spring Security)

JWT token authentication

Google OAuth login (in progress)

📩 2. Email & Notification Service

SMTP email delivery (Gmail server)

Account confirmations, job updates, recruiter notifications

Admin bulk messaging & broadcasts

Structured HTML email templates

👨‍💻 3. Job Seeker Module

Profile management (education, skills, resume upload via Cloudinary)

Job search with filters & recommendations

Real-time application tracking system

Interview scheduling integration

🏢 4. Recruiter Module

Company profile management

Job creation & posting (with skill requirements)

Application review & status updates

Candidate communication & interview scheduling

🛡️ 5. Admin Module

User & recruiter management

Job listing moderation

Payment verification & reports

System monitoring & configuration

💳 6. Payment Service

Multi-gateway: Razorpay (India) & Stripe (International)

Supports premium postings, profile upgrades, resume access

Transaction tracking, receipts & refund processing

Webhook handling for secure payment confirmation

🏗️ System Architecture

Backend: Spring Boot layered architecture (Controller → Service → Repository)
Frontend: React.js + Tailwind CSS
Database: MySQL with optimized ERD
Cloud Services: Cloudinary (file storage), SMTP (emails)
Payments: Razorpay & Stripe

📌 Database Design (ERD – Key Entities)

User: UserID, Name, Email, Role

UserProfile: Education, Skills, Resume, GitHub/LinkedIn

RecruiterProfile: Company Info, Job Listings

JobListing: Job details, requirements, postedBy

JobApplication: Application status & tracking

Payment: Transaction details

Notification: User messages & alerts
